# كيف تضيفين مقالًا جديدًا؟

1. أنشئي ملفًا جديدًا في هذا المجلد باسم إنجليزي قصير ينتهي بـ `.md`، مثال: `my-new-article.md`
   (اسم الملف يصبح رابط المقال: `/blog/my-new-article`)

2. ابدئي الملف بهذا الترويسة (Frontmatter) ثم اكتبي المقال تحتها بصيغة Markdown:

```
---
title: "عنوان المقال"
date: "2026-07-15"
lang: "ar"
excerpt: "ملخص من سطر أو سطرين يظهر في صفحة المدونة وفي نتائج البحث."
tags: ["وسم أول", "وسم ثانٍ"]
---

مقدمة المقال...

## عنوان فرعي

فقرة...

- نقطة أولى
- نقطة ثانية

**نص غامق** و *نص مائل*.
```

3. احفظي الملف. المقال يظهر تلقائيًا في صفحة /blog مرتبًا بالتاريخ (الأحدث أولًا).

ملاحظات:
- التاريخ بصيغة YYYY-MM-DD.
- اللغة: `lang: "ar"` للعربية أو `lang: "en"` للإنجليزية (تتحكم باتجاه النص وتنسيق التاريخ).
- وقت القراءة يُحسب تلقائيًا.
- هذا الملف (README) لا يظهر في المدونة.
- الأسهل: استخدمي لوحة التحكم على /admin بدل تعديل الملفات يدويًا.
